安装docker
yum install -y epel-release
yum install https://get.docker.com/rpm/1.7.1/centos-6/RPMS/x86_64/docker-engine-1.7.1-1.el6.x86_64.rpm
docker version查看

配置阿里云镜像加速
在/etc/sysconfig/docker上加上
other_args=”–registry-mirror=https://pi15f0aj.mirror.aliyuncs.com"
ps -ef|grep docker 可查看

升级内核
uname -r 查看版本
yum -y update nss
rpm –import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org
rpm -Uvh http://www.elrepo.org/elrepo-release-6-8.el6.elrepo.noarch.rpm
yum –enablerepo=elrepo-kernel -y install kernel-lt
修改/etc/grub.conf文件，将default=1修改成default=0。

镜像命令
docker images 查看本地镜像
docker search 在官方搜索
docker search -s 30 tomcat star>30的
docker pull 下载镜像
docker rmi 删除本地镜像

出现FATAL:kernel too old升级内核
容器命令
docker run -it centos
-i 以交互模式运行容器，通常与-t同时使用
-t 为容器重新分配一个伪输入终端，通常与-i同时使用
–name “容器新名字”，为容器指定一个名称
exit 进入容器后使用exit退出容器
ctrl+P+Q 容器不停止退出
容器通通删光 docker rm -f $(docker ps -q)

打开新的终端，docker ps 查看docker有哪些容器在跑
dockern run -it 加容器名称与 docker start 区别
run是要创建容器拉取镜像，start是要容器已经存在
docker restart 容器
docker stop 容器id

启动守护式容器
docker run -d centos
查看容器内运行的进程 docker top 容器id
查看容器内部细节 docker inspect 容器id
重新进入 docker attach 容器id 直接进入容器启动命令的终端，不会启动新的进程
docker exct -it 容器id bashshell 是在容器中打开新的终端，并且可以启动新的进程
从容器内拷贝文件到主机 docker cp 容器id：容器内路径 目的主机路径

容器添加数据卷两种方法
docker run -it -v /宿主机绝对路径目录：/容器内目录 镜像名

docker运行mysql
docker run -p 3306:3306 -v /zzyyuse/mysql/conf:/etc/mysql/conf.d -v /zzyyuse/mysql/logs:/logs -v /zzyyuse/mysql/data:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=123456 -d mysql

docker exec -it MySQL运行成功后的容器ID /bin/bash
mysql -uroot -p

navicat链接不了的问题
ALTER USER ‘root’@’localhost’ IDENTIFIED BY ‘password’ PASSWORD EXPIRE NEVER;这里的password是你正在使用的密码

ALTER USER ‘root’@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘password’;#更新一下用户的密码这里的password为你修改的新密码。

alter user ‘root’@’%’ identified by ‘password’ password expire never;

alter user ‘root’@’%’ identified with mysql_native_password by ‘why’;//why是自己新修改的密码。

FLUSH PRIVILEGES; #刷新权限，使自己的修改生效。

use mysql
select user,host,plugin from user where user=’root’;
2变成1

命令说明：
-p 3306:3306：将主机的3306端口映射到docker容器的3306端口。
–name mysql：运行服务名字
-v /zzyyuse/mysql/conf:/etc/mysql/conf.d ：将主机/zzyyuse/mysql录下的conf/my.cnf 挂载到容器的 /etc/mysql/conf.d
-v /zzyyuse/mysql/logs:/logs：将主机/zzyyuse/mysql目录下的 logs 目录挂载到容器的 /logs。
-v /zzyyuse/mysql/data:/var/lib/mysql ：将主机/zzyyuse/mysql目录下的data目录挂载到容器的 /var/lib/mysql
-e MYSQL_ROOT_PASSWORD=123456：初始化 root 用户的密码。
-d mysql: 后台程序运行mysql
