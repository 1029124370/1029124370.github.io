  1. 聊聊list 和 set
    集合分为 collection 和 map两个体系
    collection下有list和set
    list 是有序，可重复的集合
    set 是无序，不可重复的集合
    而这个list接口，分别有三个实现方法，分别是arraylist，linkedlist，vetor
    arraylist底层使用数组结构，线程不安全
    我的理解嗷，为什么不安全呢，是因为可能在多线程的时候一个线程会覆盖另一个线程的值，比如在多线程的时候，有a和b线程，a线程的值放进下标为0的数组，然后还没结束的时候b也在运行，然后b就覆盖了a的值，所以下标为0的位置保存了b的元素的值。
    还有一种就是放元素的时候你a线程，就是说本来你数组长度只有10，a线程b线程同时进行，然后a已经到了第10个的位置，因为数组还没有扩容，所以会出现数组越界异常

arraylist 扩容机制
就是它在构造的时候有一个有参构造和一个无参构造，无参构造就是创建一个长度为0的数组，
在有参构造里呢，创建的长度就是由参数决定的。接下来扩容的话，就是使用add方法，无参呢，就是增加数组长度到10，然后添加元素到数组里。然后它有一个ensure的方法，具体扩容就是原来的1.5倍

linkedlist底层使用链表结构，对于插入，删除操作较多，建议使用，因为它有一个头节点还有一个尾节点，头节点就是存储上一个元素的位置，尾节点就是存储下个元素的位置，所以你要增加到a和b之间的话，a的尾部节点改变，b的头部节点改变，所以速度是比较快的。
vector是安全的，底层使用数组

关于遍历的问题，使用迭代器，就是创建一个迭代器的对象，然后用hasnext的方法进行遍历
