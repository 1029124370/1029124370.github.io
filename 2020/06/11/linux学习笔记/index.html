linux学习笔记

2020-06-11
Word count: 652   |   Reading time≈ 2 min

首先要先装vmware 然后装centos运行起来，我装的是centos6
然后昨天看到的就是vmtools的安装，需要按安装压缩包，解压然后运行用终端运行并不断按确定，用了vmtools记得reboot重启

linux结构目录







xshell6链接linux服务器
linux开启sshd（终端输入setup，打开系统服务）新建会话，添加Linux服务器的IP地址，填写用户名密码
查看LinuxIP地址，打开终端输入ifconfig。

linux的用户管理 用户家目录概念
增添用户 命令useradd xm 会在指定home目录下创建一个xm文件，登录进去后按pwd会显示在当前xm文件目录下
passwd xm 然后输入指定密码

删除用户 userdel xm
删除用户+删除家目录 userdel -r xm
但除非特殊必要，一般来说不会删除家目录

查询用户
id xm
在root状态下su xm切换到xm，按exit退出
查看当前用户 whoami
clear 清理当前终端

增加组 groupadd group1
删除组 groupdel group1
增加用户直接加组 useradd -g 用户组 用户名
修改用户组 usermod -g 用户组 用户名

忘记root密码怎么办

linux运行级别

指令帮助
man help info
pwd显示当前的目录
ll查看文件列表
cd

绝对路径和相对路径

cd ../../root 退两层找同级目录root

mkdir 创建目录 mkdir -p 创建多级目录
rmdir删除空目录 如果空目录有内容那就无法删除
需要使用rm -rf删除
touch创建空文件

cp 复制指令

\cp强制覆盖不用提示

rm删除指令 rm -rf删除目录
mv移动文件或重命名

cat浏览文件（只读） cat -n 显示行号
cat 文件名｜more是分页 空格下一页 ctrl+b上一页
less也是查看文字内容，但它并不是把文件加载之后才显示，相对于more查看大型文件更有效率

指令和>>指令 指令显示的内容
覆盖 >>追加到文件的底部
echo”内容”>或>>文件
echo $path输出环境变量
head显示开头文件内容 head -n 5 显示头五行
tail输出文件后几行 默认为10行
tail -f实时监控文档更新
ln指令 也叫软连接，相当于windows的快捷方式
history查看已执行过的历史命令

date指令

cal显示日历
还有find和local指令

压缩 gzip 文件名 解压 gunzip 文件.gz
常用 压缩 zip 解压 unzip 目录压缩 zip -r
解压 tar -zxvf 文件名

查看文件所有者 ls -ahl
